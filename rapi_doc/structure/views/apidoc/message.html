GET/messages

Access:
Return: - User does not have access to this message
Parameters:
  • qrrent_id:integer - URL Param, Required

Description: Retrieves all messages in the system that the user has access to

GET/messages/new

Access:
Return: XML - A message
Parameters:

Description: Get a pre-generated, empty message

GET/messages/:id

Access:
Return: XML - The retrieved message
Parameters:
  • id:integer - Required URL param. ID of the message to retrieve

Description: Retrieve a specific message by unique id. Could return 403 | access denied; the current user does not have access to this message 404 | not found; the message requested was not found in the system

POST/messages

Access:
Return: JSON XML - The created message object
Parameters:
  • message:hash - A message object in hash form.

Description: Creates a single message given properly formatted parameters

PUT/messages/:id

Access:
Return: JSON XML - The updated message
Parameters:
  • id:integer - Required. ID of the message to update
  • message:hash - A hash of the message parameters to update to

Description: Updates a single message by unique id if given properly formatted parameters

Access:
Return: JSON XML - Response message

Description: integer | id (required): the id of the message you wish to delete Deletes a message given a specific id
